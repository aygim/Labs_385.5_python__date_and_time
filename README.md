
# Setting up a Jupyter/Python Notebook
1.  First create normal virtual env: `python(3) -m venv name_of_venv`
2.  Activate env.
3.  Install appropriate library. `pip install ipykernel`
4.  Create files with extension `.ipynb`
5.  Open file, top right `select kernel`, -> `python env`, -> select appropriate env.



-   `pip freeze > requirements.txt` saves all libraries and versions in thie venv to a test file.
